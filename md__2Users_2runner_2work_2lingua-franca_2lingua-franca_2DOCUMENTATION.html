Building Doc Files Locally

To clone the repo and build the doc files locally, simply do this:

  • Check out this repo and build the docs:
    • git clone git@github.com:lf-lang/lingua-franca.git
    • cd lingua-franca/docs
    • doxygen Doxyfile.in
